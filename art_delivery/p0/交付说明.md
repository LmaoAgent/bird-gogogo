## Cocos Creator 接入建议

1. 优先导入 `atlases/` 中对应的 PNG 与 `.plist`；若项目管线使用 JSON，则导入同名 `.json`，不要同时绑定两份元数据。
2. 门框、按钮、弹窗和进度条按 `integration_manifest.json` 的 @2x 九宫格 inset 设置为 Sliced。
3. 位图字体直接导入 `.fnt` 与同名 PNG，支持 `0–9`、`+ - × ÷`。
4. 赛道使用 `scene_marketbridge_track_tile_runtime_01.png` 纵向循环；原始 @2x 版本用于高分辨率设备或二次裁切。
5. 蒜鸟 P0 按清单允许的“静态多状态帧”交付。run/cheer/panic 已有关键姿态，开发期可先用位移、缩放、旋转和 squash/stretch 补间；正式骨骼动画再复用相同形象替换。
6. 敌人和特效的帧率、循环、混合方式及推荐 pivot 已写入 `integration_manifest.json`。

## 首包体积

`p0_startup_manifest.json` 列出首包所需三张图集、字体、运行时远景/赛道和运行时 Logo。当前合计 **2,769,229 字节（2.64 MiB）**，低于 4 MiB 上限。以下内容不计入首包：

- 1080×1920 整屏视觉基准稿；
- 1024 商店图标；
- 未压色的 @2x 场景源图；
- `source/` 中的生成原图和抠图中间图。

复验命令：

```bash

```

