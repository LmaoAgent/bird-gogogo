## 一、你要做什么
搭出《蒜鸟冲冲冲》P0 可玩原型：一支「蒜鸟大军」在竖屏赛道上狂奔，吃加成门滚雪球扩军，撞击敌方消耗其血量，血量清零则通关、兵力归零则失败。

## 三、硬约束（不可偏离）
- 引擎 **Cocos Creator**；平台 **微信小游戏**，竖屏，设计分辨率 1080×1920。
- **先做纯广告版（IAA），不做内购/商城/月卡**（版号未到，走 Plan B）。
- **数值全走配置表（JSON），不硬编码**：门、关卡、`k(level)`、`H(level)` 均可热更。
- 首包 ≤ 4MB；美术命名按 `美术需求清单.md` §0。

## 四、P0 范围（"可玩 demo"判定）
- [ ] 单指水平拖动，控制大军队形中心 X（前进自动）
- [ ] 加成门：`add / mul / sub / div / pick`，含蓝 `+1` 连排、黄 `+99`（配置见玩法文档 §3）
- [ ] 对撞模型：**结果由兑换比 `k` 决定、表现由单兵 DPS `d` 决定**的解耦设计（玩法文档 §4）
- [ ] 胜负判定 + 前 10 关配置（玩法文档 §5.2 直接用）
- [ ] 兵力队形渲染，同屏上限 `N_render`（超出用缩放+数字表现）
- [ ] 最简 HUD（当前兵力、进度）+ 通关/失败页占位
- [ ] 美术用**占位**（色块 / SVG），资源到位后按命名替换

## 五、P0 不做（放 P1）
新手引导精修、皮肤、商城、排行榜、广告接入、每日签到/任务 —— 全部 P1 再说。

## 六、建议的最小闭环（先跑通这条）
```
出兵(N0) → 拖动吃门累加 N → 撞击按 N≥H/k 判定 → 通关/失败 → 进下一关
```
先把这条闭环 + 前 3 关（必过教学关）跑起来验证手感，再铺开到第 10 关和更多门类型。
